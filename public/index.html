Python Programmable Web Server

The Programmable Server is Python-based and bare bones. It is made to have apps plugged into it.

Here is a list of what is possible without any customization:

  1. Get this screen by entering the abbreviated url localhost in your browser.
  2. Shut the server down by entering localhost/shutdown in your browser.
  3. See a bird picture by entering localhost/media/wetBird.JPG in your browser.
  4. Add other things to the public subdirectory so you can see them the way you've seen the bird picture.

A first customization might be to search for web_root in the configuration file config.py and set it to a different directory. This directory will become the root of your static file service. The default value of web_root is public which the server sees as a keyword telling it to use the default public subdirectory. When you replace public be sure to use a full path name. This path name can point to anyplace on your computer.

Note that the original configuration only allows access from browsers on the same computer as the server. (Hence the use of localhost in the above above urls rather than the more normal domain name or possible IP number.) A second customization might be to allow access for any browser that can reach your server's computer. Do this by setting localServe to False.

If you have a local network with a 255.255.255.0 mask and one gateway you can restrict access to browsers on this network by assigning the IP/4 number of the gateway to localServe.

If you are unsure what that last sentence means or whether it applies to you, don't try it! Accept that your choices are "browsers on the same computer" and "any browser that can reach your server's computer".

By the way, regardless of how you set localServe the shutdown command will only work from a browser on the same computer as the server.

A third (and recommended) customization might be to install the demonstration app which is located in the demo_app subdirectory.

The demonstration app together with its code and comments will be enough for some of you to start building your own applications. Others may want to buy the displayed ebooklet. It is available in Kindle format from Amazon and in epub format from many other sources.
