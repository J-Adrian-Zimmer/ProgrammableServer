Demo Application

This demonstration application is located in the demo_app subfolder of the Programmable Sever's directory and is described in detail in the forthcoming ebooklet “A Python Programmable Web Server”. Many of you can learn what you want to know by reading the code and comments in lieu of buying the ebooklet. Each of the three sub applications tells you where to find its code.

The application needs to be installed in the Programmable server. If you are reading this you may have already done the installation.

You can return the server to its original state by running restore.py from the server's directory.

The demos are

  1. echo which bounces text to the server and back in JSON form using AJAX technology.

  2. showRequestParameters which shows the various views of a browser request which are available through the Programmable Server.

  3. uploadPic which uses an HTML form to upload a picture.

You can invoke them with localhost\echo etc. Each one will tell you where to find its source code.
