Programmable Server
Demo Application

The Programmable Server works by executing a sequence of “expander” modules. Each of these is a Python file that chooses to respond to a particular browser request or not. Once a response has happened no more expander files are executed.

If you are reading this you have probably already installed the demonstration app. If not,

go to the demo_app subdirectory of the Programmable Server's directory and run install.py using Python 2.7.x. Then restart the server. You can return the server to its original state by running restore.py from the server's directory.

This demo application is not intended to be useful. Rather it was designed to show the variety of things a programmer of an expander can do.

There are four expanders in the demonstration:

  1. about which shows this page.

  2. echo which bounces text to the server and back using AJAX technology and formatting the transfer data with JSON.

  3. showRequestParameters which shows the various views of a browser request.

  4. uploadPic which uses an HTML form to upload a picture.

You can invoke these demonstrations with localhost\about etc. Each one will tell you where to find its source code. It is that source code and the included comments which will show you how to make your own applications.

If you need more help consider buying “A Python Programmable Web Server”.
